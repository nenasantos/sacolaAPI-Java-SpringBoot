# sacolaAPI-Java-SpringBoot
Projeto de desenvolvimento de uma API de sacola de e-commerce utilizando Java e SpringBoot.

Features
 Incluir itens na sacola
 Visualizar sacola
 Fechar sacola
 Excluir item da sacola
