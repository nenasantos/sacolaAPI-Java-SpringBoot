# sacolaAPI-Java-SpringBoot
Projeto de desenvolvimento de uma API de sacola de e-commerce utilizando Java e SpringBoot.
<br></br>

## Features:
<ul> 
 <li>Incluir itens na sacola</li>
 <li>Visualizar sacola</li>
 <li>Fechar sacola</li>
 <li>Excluir item da sacola</li>
 </ul>
